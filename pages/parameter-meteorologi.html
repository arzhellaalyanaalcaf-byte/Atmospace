Apa Itu Parameter Meteorologi?

Parameter meteorologi adalah variabel fisik atmosfer yang diukur untuk memahami kondisi cuaca dan memprediksi keadaan atmosfer di masa depan. Parameter ini diamati menggunakan berbagai instrumen seperti termometer, barometer, anemometer, hygrometer, dan sensor radiasi.

Klasifikasi Parameter Meteorologi
  • Parameter Prognostik: Parameter yang digunakan dalam persamaan pengatur atmosfer untuk memprediksi kondisi atmosfer.
  • Parameter Diagnostik: Parameter yang tidak digunakan dalam proses prediksi atmosfer, diturunkan dari parameter prognostik, atau diamati sebagai akibat dari suatu proses atmosfer.
  • Parameter Hidrometeorologi (High Impact Model): Parameter prognostik maupun diagnostik yang digunakan untuk memahami dan memprediksi kondisi hidrometeorologis.
  • Parameter Lain (Penerbangan/Pelayaran): Kondisi cuaca saat ini dan tendensinya, visibilitas, jumlah petir.
